# geofencing
Experimenting with algorithms to detect intersects from point and circles to concave/convex geofences, debugging with mapbox

## dev notes
```npm install```

```npm start```

you need a mapbox key in a .env file at the root of the project

```
MAPBOX_API_KEY=pk....
```
